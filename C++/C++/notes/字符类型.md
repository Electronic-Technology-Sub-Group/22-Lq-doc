字符类型表示一个字符，包含两种类型：

| 类型    | 占用内存（字节） | 使用场景                      | 字面量标记 | 实例      |
| ------- | ---------------- | ----------------------------- | ---------- | --------- |
| char    | 1                | 英文字符，控制符等 ASCII 字符 | ' '        | 'a', '\n' |
| wchar_t | 2                | 中文字符等 Unicode 字符       | L' '       | L'好'     |

```ad-info
`unsigned char` 类型是 Viscal C++ 定义的一个无符号类型，但不属于 `C++` 标准类型
```

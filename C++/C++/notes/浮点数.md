浮点型用于表示小数，采用 IEEE 754 标准表示法，根据其占用空间大小不同可分为以下三种，可表示的精度和值域范围也不同，使用[[float]]和[[double]]声明。

| 类型        | 占用内存（字节） | 值域        | 精度             |
| ----------- | ---------------- | ----------- | ---------------- |
| float       | 4                | ±3.4e(±38)  | 约 7 位小数准确  |
| double      | 8                | ±1.7e(±308) | 约 15 位小数准确 |
| long double | 8 或 16          |             |                  |

浮点数类型可以使用科学计数法，使用 `en` 或 `En` 表示 `10^n`。若整数部分为 0，可省略 0。

```c++
double a {1.0e5};
double b {1.0e-5};
double c {.345};
```

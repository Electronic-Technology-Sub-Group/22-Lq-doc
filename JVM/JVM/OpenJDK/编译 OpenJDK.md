# 编译环境

- 最好使用 Linux 发行版或 MacOS 便于安装依赖；Windows 下需要 Cygwin，WSL 或 MSYS2
- GCC，CLang，Visual Studio 或 Apple XCode
- 编译 JDK N 版本，至少有版本不低于 JDK N-1 的环境

JUnit  是一个 Java 平台的测试框架，JUnit 4 与 5 之间有一次很大的更新。

新版的 JUnit 5 分为三部分：
- JUnit Platform：启动框架，定义了 TestEngine API，还包括了一个单独的控制台启动器
- JUnit Jupiter：编写测试和扩展的编程模型和扩展模型，提供 TestEngine 实现
- JUnit Vintage：提供兼容 JUnit 3 和 JUnit 4 的 TestEngine 实现

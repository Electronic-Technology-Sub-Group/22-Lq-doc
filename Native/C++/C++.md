# 编译型语言？

值得注意，和流行的误解不同，ISO C 和 ISO C++ 都从未明确要求源程序被 _编译（compile）_，而仅要求 _翻译（translate）_，因此C和C++并不是所谓的 _编译型_ 语言。 摘自维基百科

# C 的超集？

一般来说，我们认为 `C++` 是 `C` 的超集，但也有例外：
- `C` 允许从 `void*` 类型隐式转换到其他指针类型，而 `C++` 不允许
- `C++` 定义了很多新关键字，如 `new`，`delete` 等，无法作为合法的标识符

将暂存区的更改记录提交到仓库，新的提交将直接继承自当前 `HEAD` 标签的提交记录。 

```bash
git commit [options]
```

- options：提交参数，至少有 `-m`，`-F` 或 `-c` 三个标签中的一个。

| 选项                        | 简写          | 作用                                       |
| --------------------------- | ------------- | ------------------------------------------ |
| `--message=<msg>`           | `-m <msg>`    | 为本提交增加一行描述，可以有多个 `-m`      |
| `--file=<file>`             | `-F <file>`   | 使用指定文件的内容作为提交描述             |
| `--resume-message=<commit>` | `-C <commit>` | 使用已存在的提交记录的描述作为本次提交描述 |

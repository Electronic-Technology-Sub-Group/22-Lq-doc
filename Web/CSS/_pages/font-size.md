可选值：
- 绝对大小：对应 `<font>` 标签的 `size` 属性（Mozilla 文档上写的相当于 1 到 7，实际测试相当于 0 到 6）

| 绝对大小   | `<font size="">` |
| ---------- | ---------------- |
| `xx-small` | 0                |
| `x-small`  | 1                |
| `small`    | 2                |
| `medium`   | 3，默认          |
| `large`    | 4                |
| `x-large`  | 5                |
| `xx-large` | 6                |

- 相对大小：相对父元素同属性值的大小，包括 `larger` 和  `smaller`，缩放比例与前面绝对大小的比例类似

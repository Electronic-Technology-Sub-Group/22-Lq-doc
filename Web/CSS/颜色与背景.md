# 背景图

使用 `background-image` 设置背景图片

![[background-image]]

**比较重要的图片应该使用 `<img>` 标签，不影响功能表达的可以使用 `background`**

# 背景复合

`background` 为背景的复合属性

```CSS
div {
    background: red url(...) repeat-x center center;
}
```

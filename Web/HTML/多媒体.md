多媒体标签用于在[[HTML#body|body]]中的特定位置显示图片，音乐或视频。

# 图片

在网页中，图片通过 `<img>` 引入。

```HTML
<img src="图片地址" alt="替换文本">
```

# 音频

使用 `<audio>` 标签插入一个音乐播放器，但默认不会显示。

```HTML
<audio src="音频文件地址"></audio>
```

![[audio]]

# 视频

使用 `<video>` 标签插入一个视频播放器

```HTML
<video src="视频文件地址"></video>
```

在 [[HTML]] 中，一行网页正文中连续多个空格只会显示一个空格，换行也不会显示。可通过 `&英文;` 的结构直接输入字符，这种形式的字符称为字符实体。

常用的字符实体有：

| 内容           | 字符实体  |
| -------------- | --------- |
| &nbsp;（空格） | `&nbsp;`  |
| &lt;           | `&lt;`    |
| &gt;           | `&gt;`    |
| &amp;          | `&amp;`   |
| &quot;         | `&quot;`  |
| &apos;         | `&apos;`  |
| &cent;         | `&cent;`  |
| &pound;        | `&pound;` |
| &yen;          | `&yen;`   |
| &euro;         | `&euro;`  |
| &sect;         | `&sect;`  |
| &copy;         | `&copy;`  |

#双标签 #表格

在[[body]]中，该标签用于

```HTML
<table>
  <caption>这是个表格</caption>
  
  <tr>
    <th>表头 1</th>
    <th>表头 2</th>
    <th>表头 3</th>
    <th>表头 4</th>
  </tr>
  
  <tr>
    <td>表 1x1</td>
    <td>表 1x2</td>
    <td>表 1x3</td>
    <td>表 1x4</td>
  </tr>
  
  <tr>
    <td>表 2x1</td>
    <td>表 2x2</td>
    <td>表 2x3</td>
    <td>表 2x4</td>
  </tr>
  
  <tr>
    <td>表 3x1</td>
    <td>表 3x2</td>
    <td>表 3x3</td>
    <td>表 3x4</td>
  </tr>
</table>
```

属性：
- border：边框[[尺寸]]
- width：宽度[[尺寸]]
- height：高度[[尺寸]]

效果：

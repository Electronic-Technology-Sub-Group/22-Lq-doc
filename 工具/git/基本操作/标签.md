`describe` 可以获取离某个提交（默认 `HEAD`）的最近的某个 tag

```bash
git describe [提交]
```

该指令返回一条字符串 `<tag>_<num>_g<hash>`
- `tag`：最近的标签名
- `num`：最近的 `tag` 与给定提交相差多少条记录
- `hash`：给定提交记录的 `hash` 的前几位

若给定提交本身有标签，则只会返回标签名

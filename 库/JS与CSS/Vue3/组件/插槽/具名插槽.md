# 具名插槽

一个控件中可以包含多个插槽实现混合渲染父组件和子组件模板

* 每个插槽使用 `name` 属性区分，使用时为元素添加 `slot=<slot-name>` 确定属于哪个插槽
* 当有多个插槽时，可以包含一个没有 `name` 的插槽，没有 `slot` 属性的元素将属于这个插槽

```html
<template id="child"><div>
    <div>
        <slot name="header">标题</slot>
    </div>
    <div>
        <slot>默认正文内容</slot>
    </div>
    <div>
        <slot name="footer">底部信息</slot>
    </div>
</div></template>
```

```html
<div id="app">
    <child-com>
        <h1 slot="header"></h1>
        <p>这是正文</p>
        <h1 slot="footer"></h1>
    </child-com>
</div>
```

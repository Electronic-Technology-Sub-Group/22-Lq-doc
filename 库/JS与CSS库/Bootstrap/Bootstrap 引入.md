# 依赖

Bootstrap 3，4 依赖于 JQuery，需要引入 Bootstrap JS 文件时应注意网页中需要包含该依赖
- [jQuery](https://jquery.com/)
Bootstrap 5 不再依赖于 JQuery，而依赖于 Popper
- [Popper - Tooltip & Popover Positioning Engine](https://popper.js.org/)

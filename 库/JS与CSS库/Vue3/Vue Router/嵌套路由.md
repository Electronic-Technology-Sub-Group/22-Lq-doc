# 嵌套路由

Vue Router 支持路由的多层嵌套，通过路由的 `children` 属性定义。

子路由：嵌套中的路由

基础路由：父层路由

* 子路由 `path` 不以 `/` 开头
* 路径为 `''` 的子路由为基础路由的默认显示

```js
const routes = [
  {
    path: '/product',
    name: 'product',
    component: ProductView,
    children: [
      {
        path: '',
        component: () => import('@/views/AlldevView.vue')
      },
      {
        path: 'alldev',
        component: () => import('@/views/AlldevView.vue')
      },
      {
        path: 'JavaEE',
        component: () => import('@/views/JavaEEView.vue')
      },
      {
        path: 'SpringBoot',
        component: () => import('@/views/SpringBoot.vue')
      }
    ]
  }
]
```

# 编程式导航

利用 `router` 或 `this.$router` 对象可在 JavaScript 中完成路由跳转，称为编程式导航

|方法名|说明|
| --------| ----------------------------------------------------|
|`push(path)`|跳转到参数指定地址|
|`replace(path)`|跳转到参数指定地址，并替换当前历史记录（无法后退）|
|`go(n)`|按历史记录前进 `n` 步，负数则为后退|
|`forward()`|前进一步，相当于 `go(1)`|
|`back()`|后退一步，相当于 `go(-1)`|

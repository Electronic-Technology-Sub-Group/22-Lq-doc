使用最广泛的 Json 序列化/反序列化组件，在插件的支持下还可以解析其他类型数据。

特点：依赖少，速度快，内存低，API 灵活
依赖：
- `jackson-core`：Json 解析和生成
- `jackson-annotations`：注解支持
- `jackson-databind`：数据绑定，依赖于前面两个包

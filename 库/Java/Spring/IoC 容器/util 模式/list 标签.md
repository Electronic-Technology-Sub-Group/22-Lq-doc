# list 标签

```xml
<util:list id="listType" list-class="java.util.LinkedList">
    <value>A simple String value in list</value>
    <value>Another simple String value in list</value>
</util:list>
```

属性：

|属性|说明|默认值|
| ------| ------------------| --------|
|`list-class`|创建的 `List` 的类型|`java.util.ArrayList`|

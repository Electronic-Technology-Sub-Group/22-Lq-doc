`QWebEngineView` 控件用于展示 HTML 页面，支持 HTML5、JavaScript 等。

```bash
pip install PyQt6-WebEngine
```

- `load(QUrl)`：加载指定 URL，支持本地 HTML 页面或某个网址
- `setHtml(QString)`：加载一段 HTML 代码

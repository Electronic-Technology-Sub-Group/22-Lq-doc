# QGroupBox

分组框，用于将一组 [[../按钮控件/QRadioButton]] 划分为一组，也可以用于细分窗口。

|属性|说明|
| ------| ------------------|
| `alignment` |对齐方式|
| `title` |分组标题|
| `flat` |是否使用扁平样式|

在函数中声明 `static` 修饰的变量，这类变量的生存期在整个程序，不会在函数返回后释放空间，其值也不会再下次调用函数时重新初始化

```run-cpp
int initialize_value() {
    cout << "initialize value" << endl;
    return 10;
}

void fun() {
    static int value { initialize_value() };
    cout << "value=" << (value++) << endl;
}

int main() {
    fun();
    fun();
    fun();
    fun();
    fun();
    return 0;
}
```

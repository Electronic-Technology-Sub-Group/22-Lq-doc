常量成员使用 `const` 修饰

* 修饰类对象，可禁止一切类成员的变更行为

```c++
class Box {
public: int value;
}

int main() {
    const Box b {1};
    // 错误，const 类无法修改其成员
    b.value = 10;
}
```

* 在尾部修饰类方法，表示该方法不会修改类成员
    * `this` 指针类型也会变成 `const ClassName*`
* `const` 修饰的类对象只能使用 `const` 修饰的方法
* 带有 `const` 的方法和不带 `const` 的方法不同

使用 `class` 关键字声明类。

```c++
class Box {
public:
    double width;
    double height;
    double length { 5.0 };
};
```

声明的类可通过 `{}` 初始化对象

```run-cpp
class Box {
public:
    double width;
    double height;
    // 默认值为 5.0
    double length { 5.0 };
    // 可使用 Box::int_type 访问
    typedef int int_type;

    void print() {
        cout << "Box: width=" << width << ", height=" << height << ", length=" << length << endl;
    }
};

int main() {
    Box b1 {};
    Box b2 {1, 2};
    Box b3 {1, 2, 3};

    b1.print();
    b2.print();
    b3.print();
    return 0;
}
```

一般来说，类的声明在头文件中，而函数实现等在源文件中。

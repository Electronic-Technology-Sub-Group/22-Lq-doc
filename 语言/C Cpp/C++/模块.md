> [!note] 使用场景
> 解决 `#include` 引入头文件存在的的某些问题

#cpp20 

模块 `module`允许将代码拆分成独立的逻辑单元，减少头文件带来的问题：
* 没有头文件
* 声明与实现可分离
* 模块名可相同，且不会冲突
* 编译后信息存储于二进制文件，提高编译效率
* 宏和未导出名称对于导入模块不可见
* 模块引入与引入顺序无关

模块在 `MSVC` 中以 `.ixx` 为后缀名，使用 `export module` 导出模块，使用 `import` 导入模块

```cpp title:test.ixx
export module cppdemo.test

export auto GetValue()
{
	return _GetValue();
}

auto _GetValue()
{
	return "Hello World!";
}
```

```cpp title:main.cpp
import cppdemo.test;

int main()
{
    cout << GetValue();
}
```

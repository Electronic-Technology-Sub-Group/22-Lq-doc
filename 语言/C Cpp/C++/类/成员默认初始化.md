#cpp11 

类的非静态成员在声明时可以使用 `=` 或 `{}` 直接初始化。

> [!warning] 初始化列表对数据成员的初始化总是优先于声明时默认初始化

```cpp
class X {
public:
    X() {}
    X(int a) : a_(a) {}
    X(double b) : b_(b) {}
private:
    int a_ = 0;
    double b_ {1.0};
}
```

#cpp20 非静态成员允许按位域初始化 ：

```cpp
struct S {
    // 低 8 位初始值为 11
    int y: 8 = 11;
    // 之后 4 位初始值为 7
    int z: 4 { 7 };
}
```

> [!note] 使用场景
> 编译时实例化类

`constexpr` 修饰类、结构体构造函数可用于编译时实例化。
* 可以有参数，参数视为常量
* 函数体必须为空
* 所在类虚函数是[[../平凡类型|平凡]]的（没有自定义析构、虚析构，所有成员对析构都是平凡的）

使用 `constexpr` 修饰的常量只能使用 `constexpr` 构造函数。带有 `constexpr` 构造函数的类称为*字面量类型（literal class type）* 。

`constexpr` 构造函数在实参非**编译时常量表达式**时，退化成普通构造函数。

#Cpp11 

将一些曾经使用[[复制构造]]的地方使用[[移动构造]]以提高效率。

```cpp
struct X;

X f(X x) {
    return x;
}

int main() {
    // C++11 以后使用移动构造而非复制构造
    X r = f(X{});
}
```

#cpp20 对右值引用和 `throw` 进行扩展。该标准规定以下情况可以采用移动代替复制：
* `return`，`co_return` 返回的对象是函数或 lambda 表达式中的对象或形参
* `throw` 抛出的对象是 `try` 块中的对象

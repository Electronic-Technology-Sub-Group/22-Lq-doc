构造函数是没有返回值，名称与类名相同的函数。

```cpp
class A {
public:
    // 任意其他构造
    A(int a) {}
    // 手动声明默认构造
    A() = default;
}
```

当一个类没有构造函数时，C++ 默认生成一个 `public` 的无参构造，实现也为空。当类中存在任何其他构造函数时，默认构造函数不会生成。

在构造函数中可以通过[[初始化列表]]完成对成员的初始化。

一个构造函数可以调用类中其他构造函数，称为[[委托构造/委托构造|委托构造]]。

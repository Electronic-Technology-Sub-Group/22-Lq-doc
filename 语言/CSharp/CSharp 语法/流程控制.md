# if-else

```csharp
if (condition1) { ... }
else if (condition2) { ... }
else { ... }
```

# switch

仅分支代码时，允许不存在 `break;`，否则产生编译错误；

使用 `goto case X;` 跳转到其他块

```csharp
switch (condition)
{
    case X: 
        // do something 
        break; 
    default: 
        // do something
        break;
}
```

# foreach

遍历 `IEnumerable<T>` 对象

```csharp
foreach (var item in items)
{
    ...;
}
```

# goto

限制：不能跳出类，不能跳出/入循环代码块，不能跳出 finally 块

```csharp
goto Label;
...;
Label: 
...;
```

# break

# continue

枚举对应整型类型和枚举类型之间可以强制转换

```csharp
enum MyEnum
{
    A,
    B,
}

(int) MyEnum.A    // 0
(MyEnum) 1        // B
```

枚举类型变量也可以与字符串互相转化：
- 返回字符串：`ToString()`
- 从字符串获取：`Enum.Parse(typeof(EnumType), "name", boolean: ignoreCase)`

# 类型转换

- `strconv` 包中包含各种基本类型与字符串之间的互相转换
- 使用 `T(v)` 将 `v` 强制转换成 `T` 类型

# 类型别名

使用 `type` 声明类型别名

```go
// 为 uint 类型产生一个别名 Age
type Age uint
```

# any

`any` 是 `interface{}` 的别名，即空接口，可以接收任意类型值

- JS 中语句末可以使用 `;` 或换行
- 一般来说，JS 按自上而下运行；例外：
	- `var` 或无声明变量提升：`var` 声明的变量提升到当前作用域最前面
	- 网页中，`alert`，`prompt` 会提前到所有 JS 和渲染之前

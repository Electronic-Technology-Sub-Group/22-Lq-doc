> [!note] 使用场景
> Rust 要求表达式总需要返回值，该类型用于实际没有返回值的表达式：
>- `continue` 进行下个循环，返回最后一个循环的结果，其他返回 `!`
>- `panic!` 宏的返回值
>- 发散函数的返回值

Rust 中有一个特殊的类型 `!`，该类型没有任何实例，也无法创建任何实例，表示不返回任何值。

由于没有任何值，所以<font color="#9bbb59">可以是任何类型</font>。

```rust
fn bar() -> ! {
    // do something
}
```

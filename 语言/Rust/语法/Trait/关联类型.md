将类型占位符与 trait​ 相关联，可以在方法签名中使用这些占位符类型，使用 type​ 声明。

> [!note]
> ​Self​​ 指向实现了当前 Trait 的类型，因此 `Self::<关联类型>` ​​ 可以用来访问关联类型。

```rust
pub struct Counter;

pub trait Iterator {
    type Item;
    fn next(&mut self) -> Option<Self::Item>;
}

impl Iterator for Counter {
    type Item = u32;
    fn next(&mut self) -> Option<Self::Item> {
        // do something
    }
}
```

SQL 语句中，关键字大小写不敏感，如 `SELECT` 与 `select` 等效；但对于具体数据大小写敏感，如 `id` 和 `ID` 表示不同的数据

若只有一条语句，句末 `;` 可省略

在 SQL 中，字符串可使用 `''` 或 `""` 包围

SQL 使用 `--` 开头表示注释

```sql title:注释
-- 这是一行注释
```

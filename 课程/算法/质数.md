# 质数表

## 埃氏筛

时间复杂度：$O(n\log\log n)$

![[v2-216bb312f6f8c9b272b329cc008c37ec_b.gif]]

```cpp
vector<int> prime;
bool is_prime[N];

void prime(int n) {
  is_prime[0] = is_prime[1] = false;
  for (int i = 2; i <= n; ++i) is_prime[i] = true;
  for (int i = 2; i <= n; ++i) {
    if (is_prime[i]) {
      prime.push_back(i);
      if ((long long)i * i > n) continue;
      for (int j = i * i; j <= n; j += i)
        is_prime[j] = false;
    }
  }
}
```

## 欧拉筛 （线性筛）

每个合数只标记一次，时间复杂度：`O(n)`

```cpp
bool isPrime[1000001];
int prime[1000001], cnt = 0;

void prime(int n)
{
	memset(isPrime, 1, sizeof(isPrime))
	isPrime[1] = 0;
	
	for(int i = 2; i <= n; i++)
	{
		if(isPrime[i])
			prime[++cnt] = i;
			
		for(int j = 1; j <= cnt && i*prime[j] <= n; j++) 
		{
			isPrime[ i*prime[j] ] = 0;
			if(i % prime[j] == 0)
				break;
		}
	}
}
```
